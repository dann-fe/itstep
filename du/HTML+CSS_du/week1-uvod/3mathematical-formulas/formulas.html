Linear equations

Linear equations can be written in this form:

ax + b = 0

Linear equations represent relationships between variables that result in straight-line graphs when plotted on a coordinate plane. They consist of terms involving variables raised to the first power and constants, enabling the expression of proportional relationships and providing a foundation for various mathematical and real-world applications. Quadratic equations

The standard form of a quadratic equation looks like this:

ax2 + bx + c = 0

Where a, b, c are known values and x is the variable or unknown. Also, a can't be 0.

Intersection

An interseciton of two sets A and B , denoted by A ∩ B, is the set of all objects that are members of boththe sets A and B. In symbols, it's

A ∩ B = (x : x ∈ A and x ∈ B)

That is, x is an element of the intersection A ∩ B if and only if x is both an element of A and an element of B.
